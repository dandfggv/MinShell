# Contributing to Minshell

Thank you for your interest in contributing to **Minshell**!  
This project is licensed under the MIT License and welcomes contributions from the community.

---

## How to Contribute

### Reporting Issues
- Use the [Issues](../../issues) tab to report bugs or request features.
- Provide clear steps to reproduce bugs.
- Include screenshots or error logs if possible.

### Suggesting Features
- Open an issue with the label `enhancement`.
- Describe the feature, why it’s useful, and how it fits into Minshell.

### Submitting Code
1. Fork the repository.
2. Create a new branch for your changes:
   ```bash
   git checkout -b feature/my-feature
